# Sistema de converção de moedas de SIFRP-TTRPG

> - SIFRP - Song Of Ice and Fire Role-Playing
> - TTRPG - Tabletop Role-Playing Game

[PT-BR] - mucho confuso, fizesmos um sisteminha pra fazer as converções

[ENG] - the system is a mess, so we decided to make a system

---

## Cobres

- Meio Vintém
- 1 Vintém = 2 Meio Vintém
- Meio Tostão = 2 Vintém
- Tostão = 4 Vinténs
- Estrela = 8 Vinténs

## Pratas

- Gamo = 7 Estrelas (ou 56 Viténs)
- Lua = 7 Gamos (ou 392 Vinténs)

## Ouro

- Dragão de Ouro (DO) = 210 Gamos

  (ou 30 Luas, ou 11.760 Viténs)

### Legenda

- Vitém de Cobre - VC
- Tostão de cobre - TC\*
- Estrela de Cobre - EC\*
- Gamo de Prata - GP
- Lua de Prata - LP\*
- Dragão de Ouro - DO

> \*Tostões de cobre (TC), estrelas de cobre (EC) e luas de prata (LP) são mais raros que viténs de cobre (VC), gamos de prata (GP) e dragões de ouro (DO)
